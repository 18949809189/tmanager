# tmanager

## tmanager 是一个Tomcat web远程管理汉化版应用

Tomcat提供了manager，可以对应用进行管理，不过manager是英文版，界面只是适配了PC端，操作上也不太流畅。因此，tmanager就出现了。


## manager包括什么
1. 基于Tomcat catalina，可以对应用进行管理（部署、卸载、停止、启动）。
2. 前端使用Amaze UI开发，适配主流浏览器，兼容移动端。
3. 支持实时查看日志
